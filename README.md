Sobre o projeto
----
O projeto foi desenvolvido no contexto do programa Oracle Next Education (ONE), em parceria com a Alura, como parte do processo de formação em desenvolvimento, um conversor de moedas desenvolvido em Java com o objetivo de realizar conversões monetárias em tempo real a partir de uma API externa de câmbio.


----------

*O sistema:*

Consome dados de uma API de taxas de câmbio

Converte valores entre diferentes moedas

Organiza o código utilizando separação de responsabilidades (controller, service, infrastructure, dto, domain)


-------------

*Tecnologias utilizadas:*

Java

Gson (para conversão de JSON)

API ExchangeRate

IntelliJ IDEA
